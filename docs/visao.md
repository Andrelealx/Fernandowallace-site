## 1. Visão Geral do Projeto

Um site-portfólio pessoal e profissional para **Fernando Wallace**, farmacêutico, acadêmico de Medicina e Secretário Municipal de Saúde de Guapimirim/RJ. O produto consolida em um único ambiente digital toda a trajetória, autoridade técnica e atuação pública de Fernando, servindo como vitrine institucional, portfólio de gestão e ponto de contato oficial.

O site não é página política. É um **portfólio de trajetória profissional e gestão pública em saúde** — elegante, confiável e humano.

### Objetivo Central
> Mostrar quem é Fernando Wallace, o que ele fez, o que ele representa e como acompanhar seu trabalho.

### Posicionamento
> **Farmacêutico, acadêmico de Medicina e gestor público dedicado ao fortalecimento da saúde em Guapimirim.**

## 4. Identidade Visual & Direção Estética

### Conceito
**"Autoridade com humanidade"** — O site une a seriedade de um gestor público com a acessibilidade de quem veio da saúde e cuida de pessoas. Não é frio como governo, nem informal como Instagram. É o ponto exato entre **institucional e humano**.

### Paleta de Cores
```
Azul-marinho:   #0B1F3A  (base dominante — confiança, autoridade, seriedade)
Verde saúde:    #1FA67A  (acento primário — cuidado, SUS, vida, farmácia)
Verde escuro:   #0F5F4A  (hover/variação do verde)
Branco gelo:    #F7FAFC  (fundo principal — limpeza, saúde)
Cinza texto:    #4A5568  (corpo de texto — legível, moderno)
Dourado suave:  #C9A646  (acento de destaque — liderança, reconhecimento)
```

### Tipografia
```
Display / Títulos:  "Playfair Display" (serif elegante, autoridade sem arrogância)
Subtítulos:         "Playfair Display" Italic
Corpo de texto:     "Source Sans 3" (humanista, altamente legível)
Labels / Dados:     "JetBrains Mono" (para datas, cargos, números)
```

### Tom de Comunicação
- Primeira pessoa moderada ("Minha trajetória na saúde pública...")
- Linguagem acessível — cidadãos comuns devem entender
- Sóbrio mas próximo — não parece panfleto político
- Foco em fatos comprováveis e registros públicos
- Valores: compromisso, cuidado, evolução, presença, responsabilidade

## 8. Stack Técnica Recomendada

### Opção A — HTML/CSS/JS Puro (Recomendada para primeira entrega)
```
Tecnologia:    HTML5 + CSS3 + Vanilla JS
Hospedagem:    Hostinger (plano compartilhado)
Formulário:    Formspree ou EmailJS
Analytics:     Google Analytics 4 + Search Console
Deploy:        FTP via Hostinger ou Git
```
✅ Rápido de entregar, leve, sem custo de infra
⚠️ Atualização de conteúdo requer edição de HTML — ok para primeira fase

### Opção B — Next.js + CMS (Evolução futura)
```
Frontend:      Next.js 14 (App Router)
Estilo:        Tailwind CSS
CMS:           Sanity.io (painel para Fernando ou assessoria atualizar)
Formulário:    React Hook Form + Resend
Deploy:        Vercel
```
✅ Fernando ou assessoria podem adicionar notícias sem programar
⚠️ Mais complexo, pequeno custo mensal

### Recomendação Leal Systems
> Entregar Opção A com qualidade máxima. Se Fernando quiser atualizar notícias com frequência → migrar para Opção B com CMS Sanity.
